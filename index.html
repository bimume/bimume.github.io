2019 流水账

其实每年都怕写这个主题的，毕竟感觉每年都一样，没啥区别

回顾一下2018的目标

  • 找一个自己喜欢的项目，然后尽全力的去做–达成！
  • 多回家，不要和家人发生任何矛盾，多表达，多沟通–未达成
  • 学通FCPX，争取剪辑6个以上完整的游戏短片–未达成
  • 学习新媒体或者Amazon的运营–已废弃
  • 精简生活（不冲动买游戏、不冲动买数码产品）–达成

就像我在Instagram上感叹的：尽了力，不内疚，出了锅，就背着。第一项还不错，虽然在我最用心的活动上还是翻车了，小年夜是真的痛。

某GV项目进行到大理后，正要回广州的当天，爸爸和我说了奶奶过世的消息，回了家，人生第一次葬礼。和家人的沟通上还是不多，如果四个字来形容那就是“有问有答”？不然弟弟结婚的事情，也不致于会让我震惊吧。

剪辑技巧都是之前掌握的，依旧没有系统化的学习，且我2019居然只发了一个视频？？？why？？？我也没打多少游戏啊。

虽然也买了大件，但是年底前都出了。

2019的小目标算是55开，比往年都有进步。

2019最开心的事情，应该是入职了现在的公司，认识了很多优秀的小伙伴，且与他们一起共事。因为他们的影响，也使得我想变得优秀。

2020的小目标

  • 工作上得到小伙伴的认可、领导的赞同
  • 皮肤/身材管理—小伙伴能一一一一一眼看出区别的那种
  • 去2个以上的国家一一一一一一一一 再不济，泰国走一圈
  • 英语或者日语系统学习一一一一一一能拿下一两个证书那就好了
  • 继续精简生活一一一一一一一一一一不乱买游戏、不冲动买消费产品
  • （如果这条实现了，我就写出来，不然就烂在肚子里吧）

年会

开场，和一群美少女合照，当然也有我喜欢的弟弟；

会中，和tim弟弟敬了帮助我们的小伙伴，其他组的小伙伴也客气了一波，大老远的跑来敬我们；

end，我的iPhone 11 Pro没抽到，抽到了体脂秤，应该庆幸没有陪跑。

感悟：吃得真的太垃圾了，年会就是大型的网友面基现场。

工作

出差。2019或许是我这辈子出差最多的一年吧，去了很多城市。北京、广州、天津、石家庄、成都、德宏、珠海……见了GV里很多恶心的人。索性爱笑的胖子运气真的好，出差的日子里只遭遇过1次飞机延误。出差办的事情也比较顺利。这个环节可以说是挺不错的。2020年会怎样呢？

办公室。没遇上恶心的人和事，不过发现微信被拉黑了，还在反思哪里没做好。

同事。结交了几个小伙伴，一起吃肉、一起吐槽互联网、一起八卦公司。尴尬的时候有人收留住宿、不懂规则的时候可以听他们的建议、需要外部支援的时候能告知找谁。已经挺好了。

生活

我记得刚到广州的那3个月，好想找时间记录下各种不习惯、不舒服。怎么现在回忆起来，好像一切都蛮easy的，广州，真香。

再战黑苹果-i5-6500-RX580-B150M-D3H

上次装的稳定用了半年有余，结果周末装windows的时候，格错盘了。

今天安装完成后，简单记录下过程。

下载镜像

我用的是苹果小兵的镜像

用 TransMac 制作macOS安装USB盘

Restore with Disk Image 点几下OK，然后等。

利用微PE挂载EFI分区，然后替换USB盘里的EFI

这里的前提就是，你需要找到适合你自己的EFI文件（远景、tonyx86）下载微PE

EFI下载

国外：https://www.dropbox.com/s/yxf9gjj4tmiwlq6/EFI.7z?dl=0

国内：https://share.weiyun.com/5TjZhwZ

设置B150M-D3H主板BIOS

【Peripherals】-USB Configuration
Legacy USB Support-> Enable
XHCI Hand-off-> Enable

【Peripherals】-Super IO Configuration
Serial Port-> Disable

【Chipset】
VT-D —> Disable
IOAPIC 24-119Entries —> Disable
Intertnal Graphics —> Disable
—-下方忽略—-
DVMT预载内存大于等于64M
Chipset →Integrated Graphics : Enabled
Chipset → DVMTPre-Allocated :128M
▪ 使用A卡或者N卡
                Peripherals →Initial Display Output : PCIe 1 Slot
▪ 仅使用核显
                 Peripherals →Initial Display Output : IGFX

不会设置的可以参考这个链接

安装Mac

开机按F12键进入引导管理,选择macOS安装USB盘,回车。

进到Clover后，选择Boot OS X Install fromXXXXX，抹盘、直到安装完成。

进入到系统后，安装 Clover Configurator 挂载EFI分区，再次把EFI拷贝到硬盘上的EFI分区。

添加UEFI引导选项

还是用的微PE，在步骤三有截图。

具体的步骤，可以参考小兵的教程

个性化设置、使用

比如设置鼠标的滚动模式，安装一些常用软件。

一切完成后，记得用 Kext Utility 清理。重启，投入日常使用。
